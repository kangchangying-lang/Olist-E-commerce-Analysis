# Olist E-commerce Operational Analysis

## Project Overview
This project analyzes the operational performance of Olist (a Brazilian e-commerce platform) from 2017 to 2018. The focus is on measuring business scalability and identifying root causes of order cancellations.

## Key Visualization: Operational Health Dashboard
![Operational Dashboard](./dashboard_20260519.png)

## Key Insights
* **Scalability**: Order volume grew 10x while cancellation rates remained stable.
* **Resilience**: Black Friday peaks showed no disproportionate spike in cancellation rates.
* **Maturity**: Operational efficiency improved in early 2018 compared to 2017.

* **Libraries**: Pandas, Matplotlib, Seaborn
